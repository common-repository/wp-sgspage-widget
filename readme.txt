WP-SGSPage Widget is a per page text and subpage menu widget. It's title and content
can be changed by the page that is currently displayed in the frontend.

== Description ==
WP-SGSPage Widget is a per page widget. That means the WP-SGSPage Widget's title and content can
be changed by the page that is currently displayed in the frontend.



== Installation ==
1. Upload *wp-sgspagewidget* folder to the */wp-content/plugins/* directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add *WP-SGSPage Widget* to your widgetable area and specify the (default) title and (default) content.
4. Create a new page or edit an existing.
5. At the bottom of the page editor you will finde the *SGSPage Widgte* box. If not, click the *Option* tab
   at the top of the editor page and enable it by checking the SGSPage Widget checkbox.
   Specify the title and content (text widget) of which is to be displayed if this page is the current in the frontend.
